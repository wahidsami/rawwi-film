# v3 Shared Operational Rules

- Use exact evidence snippets only.
- Keep findings narrow and title-only.
- Do not surface article numbers or atom numbers in titles.
- Prefer the most specific category that clearly owns the evidence.
- If another pass already owns the span, do not duplicate it here.
- If the case is unclear, return no finding.
- Never infer speaker, target, or relationship unless the snippet states it plainly.
- Do not explain the whole story; explain only the text in front of you.
