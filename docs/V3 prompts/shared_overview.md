# v3 Shared Operational Rules

- Use exact evidence snippets only.
- Keep findings narrow and title-only.
- Do not surface article numbers or atom numbers in titles.
- Prefer the most specific category that clearly owns the evidence.
- If another pass already owns the span, do not duplicate it here.
- If the case is unclear after checking the nearby scene/story memory, return no finding.
- Use nearby scene/story memory to identify speaker, target, relationship, and narrative treatment when the evidence snippet is short.
- Do not invent speaker, target, or relationship when neither the snippet nor nearby scene/story memory establishes it.
- Do not explain the whole story; explain only the text in front of you.
- Do not promote a vague insult, curse, accusation, or complaint into a religious, historical, or political finding unless the text itself clearly names that domain.
- If the snippet could fit a more direct label like profanity, bullying, family values, parents, women, or elderly, prefer that direct label over broader thematic labels.
