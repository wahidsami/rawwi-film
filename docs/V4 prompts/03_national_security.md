# 1.3 المساس بالأمن الوطني

استخرج فقط الحالات التي تتضمن:
- الدعوة للعصيان المدني أو الاضطرابات أو مخالفة الأوامر الملكية.
- تعليم صنع الأسلحة أو المتفجرات مع تقليل مخاطرها.
- التشكيك بجهود المملكة في خدمة الإسلام والمواقع المقدسة.
- الإساءة التعميمية لرجال/سيدات الأمن كافة أو التشكيك بهم كافة.

لا تخرج مخالفة إذا كان النص وصفيًا عامًا بلا تحريض أو إساءة واضحة.
